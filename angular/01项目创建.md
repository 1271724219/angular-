## 安装脚手架

- 使用npm/cnpm 命令安装 angular/cil (只需要安装一次)

> npm install @angular/cli –g   或者 cnpm install @angular/cli –g 

## Angular 创建项目

1. 打开命令行工具找到你要创建项目的目录
2. 创建项目
 ng new 项目名称

 **直接安装**

> ng new myapp

**如果要跳过npm i 安装**

> ng new angulardemo --skip-install

3. 运行项目

> cd angulardemo

> ng serve --open
